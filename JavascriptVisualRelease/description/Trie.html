是什么？

     前缀树

     当有某个节点的值不存在时，使用空心节点表示（与Radix tree不同，它会将两个字符放在一个节点） 比如：输入de字符串，此时在Trie里会出现d和e两个节点，其中的d节点是空心节点；而在Radix则会使用一个节点存储de字符串。如果后续再输入d字符串，则Trie会将d实心化，Radix则会将de节点拆分为d和e两个节点。

如何表示？

     type Trie struct { Root *TrieNode } type TrieNode struct { Childs map[rune]*TrieNode Exist bool // 是否真实存在 Data string IsLeaf bool }
